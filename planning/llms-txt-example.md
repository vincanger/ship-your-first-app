# Ship Your First App

> Learn web development by building real apps with AI.
> Point your Claude Code session at any module below to start learning.

## How to use this course
1. Install Wasp (https://wasp-lang.dev) and Claude Code
2. Run: wasp new my-app && cd my-app && wasp start
3. Open Claude Code in your project
4. Tell Claude: "Fetch and follow the instructions at [this-site]/module-0.md"
5. Learn by building!
